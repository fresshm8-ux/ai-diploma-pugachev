# Sales Analytics Project

## 📊 Описание проекта
Проект для анализа продаж с использованием SQLite и Python. Включает создание базы данных, генерацию тестовых данных и выполнение аналитических запросов.

## 🚀 Технологии
- Python 3.8+
- SQLite3
- Git
- Tabulate (для форматированного вывода)

## 📁 Структура проекта
```bash
sales-analytics/
├── main.py # Главный скрипт с анализом
├── database/
│ └── sales.db # SQLite база данных
├── reports/
│ └── analytics_report.txt # Сгенерированный отчет
├── requirements.txt # Зависимости
└── .gitignore # Игнорируемые файлы
```

## 🗄️ Структура базы данных

### Таблица `products`
- `id` (PRIMARY KEY) - уникальный идентификатор
- `name` (TEXT) - название товара
- `category` (TEXT) - категория товара
- `price` (REAL) - цена товара

### Таблица `customers`
- `id` (PRIMARY KEY) - уникальный идентификатор
- `name` (TEXT) - имя покупателя
- `email` (TEXT) - email покупателя
- `city` (TEXT) - город покупателя

### Таблица `sales`
- `id` (PRIMARY KEY) - уникальный идентификатор
- `product_id` (FOREIGN KEY) - ссылка на товар
- `customer_id` (FOREIGN KEY) - ссылка на покупателя
- `sale_date` (TEXT) - дата продажи
- `quantity` (INTEGER) - количество товара
- `total_amount` (REAL) - общая сумма продажи

## 📈 Аналитические запросы

1. **Выручка по категориям товаров** - анализ прибыли по категориям
2. **Топ-10 покупателей** - определение лучших клиентов
3. **Динамика продаж по месяцам** - сезонность и тренды
4. **Средний чек по городам** - географический анализ
5. **Самые популярные товары** - анализ спроса

📝 Особенности реализации
Форматирование чисел
Все денежные суммы форматируются с разделителями тысяч для удобства чтения:

6383000 → 6,383,000

859400 → 859,400

Автоматическое создание БД
Если база данных не существует или пуста, скрипт автоматически:

Создает все необходимые таблицы

Заполняет их тестовыми данными

Генерирует 300+ случайных продаж

🐛 Возможные проблемы и решения
Ошибка "module 'tabulate' not found"
bash
pip install tabulate
База данных не создается
Убедитесь, что у вас есть права на запись в директории проекта.
